# Privacy Policy – Pomodoro Clock

Last updated: December 2025

Pomodoro Clock is a simple productivity-focused Chrome extension. Your privacy is important, and this extension is designed to collect **no personal or sensitive user data**.

## Data Collection
Pomodoro Clock does not collect, transmit, or sell any personal data.

## Local Storage
The extension uses Chrome’s local storage only to save:
- Focus and break duration preferences
- Timer state (focus/break)
- User-selected settings

All data is stored locally on the user’s device and is never transmitted externally.

## Notifications
The extension uses notifications only to inform users when a focus or break session has ended.

## Permissions
Pomodoro Clock requests only the minimum permissions required for core functionality:
- Storage (to save preferences)
- Alarms (to manage timer events)
- Notifications (to alert session completion)

No browsing data, website content, or personal identifiers are accessed.

## Third-Party Services
Pomodoro Clock does not use any third-party analytics, tracking, or advertising services.

## Changes
This privacy policy may be updated if functionality changes. Any updates will be reflected on this page.

## Contact
If you have questions about this privacy policy, you may contact the developer via the Chrome Web Store listing.
